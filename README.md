# 📈 Stock Portfolio Tracker

A simple Python console application that allows users to purchase products, calculate the total bill, and save the final bill to a text file.


## 📖 Project Overview

The Stock Portfolio Tracker is a beginner-friendly Python console application that simulates purchasing products from a predefined list. It allows users to select multiple products, calculates the total cost, generates a formatted bill, and stores the bill in a text file. This project was developed to strengthen fundamental Python programming concepts such as dictionaries, loops, conditional statements, lists, tuples, and file handling.

## 🚀 Features

- Display available products and their prices
- Purchase multiple products in a single session
- Case-insensitive product search
- Quantity validation (prevents invalid quantities)
- Calculate the grand total automatically
- Generate a formatted final bill
- Save the bill to `portfolio.txt`

## 🛠 Technologies Used

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the program:

```bash
python stock_portfolio_tracker.py
```

4. Follow the on-screen instructions to:
   - Enter the product name.
   - Enter the quantity.
   - Continue purchasing products or type `exit` to finish.

5. The final bill will be displayed on the screen and saved automatically as `portfolio.txt`.
